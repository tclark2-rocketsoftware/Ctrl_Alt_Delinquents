# Ctrl_Alt_Delinquents Polling Web App

## Project Overview
Ctrl_Alt_Delinquents is an advanced polling web application designed to facilitate interactive polls, voting, and analytics. The project leverages AI coding agents for rapid development and feature expansion.

## Key Features
- User authentication (sign up, login)
- Poll creation, editing, and deletion (admin)
- Multiple poll types (single-choice, multiple-choice, ranked-choice)
- Real-time results and analytics
- Commenting and discussion on polls
- Scheduling and expiry of polls
- Notifications for users
- Admin dashboard for management
- Mobile-friendly responsive UI

## Actors & Use Cases
### User
- Register/Login
- View Polls
- Vote in Poll
- View Results
- Comment on Poll
- Receive Notifications

### Admin
- Create Poll
- Edit Poll
- Delete Poll
- View Analytics
- Manage Users

## Technology Stack
- Python (backend)
- Web framework (e.g., Flask or Django)
- Frontend: HTML, CSS, JavaScript (React or similar)
- Database: SQLite/PostgreSQL
- Websockets/AJAX for real-time updates

## Folder Structure
- `src/` - Main application code
- `tests/` - Unit and integration tests
- `Documentation/` - Project documentation
- `requirements.txt` - Python dependencies

## Getting Started
1. Set up Python environment
2. Install dependencies from `requirements.txt`
3. Run the application using `python src/main.py`
